Flash the latest ESP32 firmware directly from your browser.

Install firmware directly using USB.

🚀 Fast & Easy

🔒 Safe Firmware

How To Install

  1. Connect your ESP32 board using USB
  2. Click the Install button above
  3. Select the correct COM Port
  4. Wait for flashing to complete
  5. Enjoy your firmware 🚀
